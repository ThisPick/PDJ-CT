# 🎓 ระบบจัดการและประเมินผลโครงงานนักศึกษา (STD-UTC)
> เอกสารอธิบายสิทธิ์การเข้าใช้งานและคู่มือการติดตั้งระบบสำหรับนักพัฒนา

---

## 👥 1. สิทธิ์การใช้งานในระบบ (Role-based Permissions)

ระบบใช้การตรวจสอบสิทธิ์ผ่าน **JWT Token** โดยแบ่งระดับการเข้าถึงดังนี้:

### 🟢 นักศึกษา (Student)
* [x] **Dashboard:** ติดตามสถานะโครงงานของตนเอง
* [x] **Project Submission:** ยื่นเสนอหัวข้อและอัปโหลดไฟล์ PDF (ข้อเสนอโครงงาน)
* [x] **Archive:** ค้นหาและดูข้อมูลโครงงานที่ได้รับอนุมัติแล้ว
* [x] **Profile:** จัดการข้อมูลส่วนตัวและรูปภาพ

### 🟡 ครู / อาจารย์ (Teacher)
* [x] **Admin Dashboard:** ดูสถิติและภาพรวมโครงงานในแผนก
* [x] **Project Management:** ตรวจสอบรายละเอียดโครงงานนักศึกษาทั้งหมด
* [x] **Evaluation:** ให้คะแนนและประเมินผลตามเกณฑ์ Rubrics
* [x] **Milestone:** ติดตามความคืบหน้าตามระยะเวลาที่กำหนด

### 🔴 หัวหน้าแผนก (Department Head)
* [x] **Full Access:** เข้าถึงสิทธิ์ของอาจารย์ได้ทั้งหมด
* [x] **Approve System:** ตัดสินผลการอนุมัติหัวข้อโครงงาน (Approve/Reject)
* [x] **User Management:** เพิ่ม/ลบ และจัดการสิทธิ์ผู้ใช้งานทุกคน
