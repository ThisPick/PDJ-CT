🎓 ระบบจัดการและประเมินผลโครงงานนักศึกษา (Student Project Management System)
ระบบนี้ถูกออกแบบมาเพื่อจัดการโครงงานของนักศึกษาอย่างเป็นระบบ รองรับการส่งงาน ตรวจสอบ ประเมินผล และจัดการผู้ใช้งาน โดยมีการแบ่งสิทธิ์การเข้าถึงอย่างชัดเจน

👥 สิทธิ์การใช้งานในระบบ (Role-based Permissions)
ระบบจำกัดสิทธิ์การเข้าถึงผ่าน ProtectedRoute.jsx (หน้าบ้านเช็คผ่าน localStorage) และตรวจสอบ Token ฝั่งเซิร์ฟเวอร์เพื่อความปลอดภัย โดยแบ่งผู้ใช้เป็น 3 กลุ่มดังนี้:

🎓 1. นักศึกษา (Student)
เป็นกลุ่มผู้ใช้งานทั่วไปที่เน้นการส่งและติดตามผลงานของตนเอง
StudentDashboard: ดูภาพรวมสถานะโครงงานของตนเองและสถิติต่างๆ
Projectsubmit: ยื่นแบบเสนอโครงงาน อัปโหลดไฟล์ PDF และแก้ไขข้อมูลโครงงาน
ProjectArchive: ดูคลังโครงงานรุ่นพี่หรือโครงงานอื่นๆ ที่ได้รับอนุมัติแล้ว
Studentprofile: จัดการข้อมูลส่วนตัวและเปลี่ยนรูปโปรไฟล์

👨‍🏫 2. ครู / อาจารย์ (Teacher)
เป็นกลุ่มผู้นิเทศหรือผู้ตรวจการเบื้องต้น มีสิทธิ์จัดการเนื้อหาทางวิชาการ
AdminDashboard: ดูสถิติรวมของนักศึกษาทุกคนในแผนก
ProjectSTD: ดูและจัดการรายชื่อโครงงานทั้งหมด
EvaluationPage: ประเมินและให้คะแนนโครงงานตามเกณฑ์
Milestone: ติดตามความคืบหน้าของแต่ละโครงการตามระยะเวลา
RubricSettings: ตั้งค่าและแก้ไขเกณฑ์การให้คะแนน (Rubrics)

👑 3. หัวหน้าแผนก (Department Head)
เป็นกลุ่มที่มีอำนาจสูงสุดในระบบ (Super Admin)
สิทธิ์พื้นฐาน: ครอบคลุมสิทธิ์การใช้งานของ Teacher ทั้งหมด
ApproveProject: อนุมัติหรือไม่อนุมัติหัวข้อโครงงาน พร้อมให้ Feedback
ManageUsers: เพิ่ม, ลบ, แก้ไข หรือระงับการใช้งานบัญชีผู้ใช้ทุกคนในระบบ

🚀 วิธีติดตั้งและเริ่มต้นใช้งานโปรเจกต์ (Getting Started)
การรันโปรเจกต์นี้ในเครื่องของคุณ จะต้องแบ่งการทำงานออกเป็น 2 ส่วน คือ ฝั่งหลังบ้าน (Backend) และ ฝั่งหน้าบ้าน (Frontend)
⚙️ สิ่งที่ต้องมีในเครื่องก่อนเริ่ม (Prerequisites)
Node.js (เวอร์ชัน 16 ขึ้นไป)
PostgreSQL (สำหรับฐานข้อมูล)
Git
🛠️ ขั้นตอนที่ 1: การตั้งค่าฐานข้อมูล (Database)
เปิดโปรแกรมจัดการฐานข้อมูล (เช่น pgAdmin หรือ DBeaver)
สร้าง Database ใหม่ชื่อว่า std_utc (หรือชื่อตามที่คุณกำหนด)
นำเข้าไฟล์ SQL (ถ้ามีไฟล์ .sql สำหรับสร้างตาราง) เพื่อสร้างโครงสร้างตารางผู้ใช้และโปรเจกต์

🔌 ขั้นตอนที่ 2: การตั้งค่าหลังบ้าน (Backend)
เปิด Terminal แล้วเข้าไปที่โฟลเดอร์ Backend:
Bash
cd backend
ติดตั้งไลบรารีที่จำเป็น:
Bash
npm install
สร้างไฟล์ .env ไว้ในโฟลเดอร์ backend และกำหนดค่าการเชื่อมต่อดังนี้:

ข้อมูลโค้ด
PORT=5000
DB_USER=postgres
DB_PASSWORD=รหัสผ่านpostgresของคุณ
DB_HOST=localhost
DB_PORT=5432
DB_NAME=std_utc
JWT_SECRET=ใส่รหัสความลับอะไรก็ได้เช่น123456
สั่งรันเซิร์ฟเวอร์:

Bash
npm run dev
(เซิร์ฟเวอร์จะรันที่ http://localhost:5000)
💻 ขั้นตอนที่ 3: การตั้งค่าหน้าบ้าน (Frontend - React/Vite)
เปิด Terminal แท็บใหม่ แล้วเข้าไปที่โฟลเดอร์ Frontend:

Bash
cd frontend
ติดตั้งไลบรารีที่จำเป็น:

Bash
npm install
สั่งรันหน้าเว็บ:

Bash
npm run dev
(หน้าเว็บจะเปิดขึ้นมาที่ http://localhost:5173)
